# Firmware

ESP32 firmware for receiving binary LED frames over WebSocket and driving WS2811 LEDs.

Project path:
- `firmware/firmware_esp32c6_ws2811`

Toolchain requirements:
- `firmware/toolchain-requirements.md`

Build and flash instructions (from scratch, including ESP-IDF setup):
- `firmware/firmware_esp32c6_ws2811/README.md`

Important:
- Keep `sdkconfig` local; it may contain Wi-Fi credentials.
- Commit only `sdkconfig.defaults` for shareable defaults.
- `managed_components/` is generated by ESP-IDF and should stay out of git.
